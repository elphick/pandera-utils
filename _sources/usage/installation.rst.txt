Installation
============

.. toctree::
   :maxdepth: 2
   :caption: Installation:


The package is pip installable.

..  code-block::

    pip install pandera-utils

or, if poetry is more your flavour.

..  code-block::

    poetry add pandera-utils

or you may already be on the uv bus!

..  code-block::

    uv add pandera-utils
